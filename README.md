# vMix Pro

## Overview

vMix Pro is a software application designed for live video production and streaming on Windows systems. It enables users to mix, switch, record, and stream multiple video sources in real-time, supporting a range of inputs and outputs for various production needs.

## Key Features

**Real-time video mixing and switching**  
**Support for multiple video inputs including cameras, videos, images, and audio**  
**Multi-bitrate streaming to various platforms**  
**Recording of live productions in high-quality formats**  
**Customizable overlays, titles, and effects**  
**Audio mixing with advanced controls**  
**Integration with external hardware for enhanced control**  
**Support for virtual sets and chroma key effects**

## Why vMix Pro?

vMix Pro provides a stable environment for video production with a focus on clear and reliable operation.
